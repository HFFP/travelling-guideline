# 深圳 - 云南

## 行程安排

### 深圳-阳朔兴坪
* 行：525km,车程6小时
* 吃：桂林米粉
* 住：贵的房间风景好可看双峰，或者房间普通公共区域风景好
* 玩：逛古镇，相公山（套票180，包船车门票）
